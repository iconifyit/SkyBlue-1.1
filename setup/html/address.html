Enter Your Web Site Address (URL)

Enter your web site address below. The value you enter should be a fully qualified domain, including a the sub-directory if SkyBlueCanvas is being installed in a sub-directory in your web root. For example, if your domain is www.mydomain.com and you are installing SkyBlueCanvas in a sub-directory named 'skyblue', you need to enter www.mydomain.com/skyblue/.
